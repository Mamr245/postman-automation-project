<h1 align="center">Postman Automation Project :computer:</h1> 

### :question: What is this?
Automated API testing workflow using Postman to validate the functionality (GET, POST, PUT, DELETE) of https://restful-api.dev/. The project also includes a GitHub Actions workflow for running the test suite. You can view past runs and results [here](https://github.com/Mamr245/postman-automation-project/actions).

### :hammer: Technologies used:
`Postman`
